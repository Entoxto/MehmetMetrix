# Инструкция по добавлению товаров в каталог

## Структура данных

Все товары хранятся в файле `data/products.json`.

## Как добавить новый товар

1. Откройте файл `data/products.json`
2. Добавьте новый объект в массив `products`:

```json
{
  "id": "fur-003",
  "name": "Название товара",
  "category": "Мех",
  "photo": "/images/products/fur-003.jpg",
  "sizes": ["S", "M", "L", "XL"],
  "price": 25000,
  "description": "Описание товара",
  "materials": {
    "outer": "Верхний материал",
    "lining": "Состав подкладки",
    "comments": "Комментарии по материалам"
  },
  "inStock": true,
  "tags": ["зима", "премиум"]
}
```

## Как добавить фото

1. Поместите фото в папку `public/images/products/`
2. Назовите фото так же, как указано в поле `photo` в JSON (например, `fur-003.jpg`)
3. Рекомендуемый формат: JPG или PNG
4. Рекомендуемый размер: 1200x1600px или больше

## Поля товара

- **id** - уникальный идентификатор (например, "fur-001")
- **name** - название товара
- **category** - категория: "Мех", "Замша", "Кожа" или "Экзотика"
- **photo** - путь к фото (начинается с `/images/products/`)
- **sizes** - массив доступных размеров
- **price** - цена в долларах (опционально)
- **description** - описание товара (опционально)
- **materials.outer** - верхний материал (опционально)
- **materials.lining** - состав подкладки (опционально)
- **materials.comments** - комментарии по материалам (опционально)
- **inStock** - наличие на складе (true/false)
- **tags** - теги для поиска/фильтрации (опционально)

## Примеры названий фото

Если в JSON указано `"photo": "/images/products/fur-001.jpg"`, то файл должен называться `fur-001.jpg` и лежать в папке `public/images/products/`.
